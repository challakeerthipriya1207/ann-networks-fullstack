## 📖 Overview
ANN Networks is a fully responsive, full-stack web application designed to manage and display job listings and professional blog articles. Featuring a secure admin dashboard, it allows for real-time database updates and seamless content management.

## 💻 Tech Stack
* **Frontend:** React.js (Hosted on Vercel)
* **Backend:** Node.js, Express.js (Hosted on Render)
* **Database:** MySQL (Hosted securely on Aiven Cloud)
* **Security:** JSON Web Tokens (JWT) for authentication, CORS policy enforcement

## ✨ Key Features
* **Public Portal:** Visitors can seamlessly browse the latest job openings and technical blog articles.
* **Secure Admin Dashboard:** Protected by JWT authentication to prevent unauthorized access to backend operations.
* **Dynamic Data Management:** Admins can Create, Read, Update, and Delete (CRUD) job postings and articles directly from the UI.
* **Cloud Integration:** Fully deployed with cloud database hosting to ensure 24/7 uptime and data persistence.
